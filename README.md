# Expense Tracker

**Автор:** Ренат Адамов

## Описание
Приложение для учёта личных расходов с графическим интерфейсом. Позволяет добавлять траты, фильтровать по категории и диапазону дат, а также автоматически подсчитывать итоговую сумму за выбранный период. Все данные сохраняются в JSON-файл.

## Как использовать
1. Установите Python 3.6+.
2. Запустите `expense_tracker.py`:
markdown

text
3. В верхней панели введите сумму, выберите категорию, укажите дату и нажмите **«Добавить расход»**.
4. Для фильтрации укажите категорию и/или диапазон дат, затем нажмите **«Применить фильтр»**.
5. Кнопка **«Сбросить фильтр»** показывает все записи.
6. Сумма за текущий отфильтрованный период отображается внизу.

## Примеры использования
| Действие | Результат |
|----------|-----------|
| Добавить «100» Еда 2025-05-01 | Появляется запись в таблице |
| Добавить «200» Транспорт 2025-05-02 | Появляется вторая запись |
| Фильтр: Еда, с 2025-05-01 по 2025-05-01 | Показывает только 100, сумма = 100 |
| Фильтр сброшен | Показывает обе записи, сумма = 300 |
